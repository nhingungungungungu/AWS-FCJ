Đây là nhật ký công việc chi tiết của chương trình **First Cloud Journey (FCJ) Workforce** được thực hiện trong **12 tuần** (84 ngày). Mỗi tuần tập trung vào một khía cạnh khác nhau của AWS Cloud, từ nền tảng cơ bản đến các dịch vụ nâng cao, giúp xây dựng kiến thức vững chắc cho vai trò **AWS Cloud Engineer**.

### Lộ trình học tập 12 tuần:

**Tuần 1:** [Nền tảng AWS - Làm quen với Account, IAM, VPC và EC2](1.1-week1/)
- Tạo tài khoản AWS, cấu hình IAM, hiểu về VPC và EC2 cơ bản

**Tuần 2:** [Bảo mật và Mạng lưới - Security Foundations & Networking](1.2-week2/)
- Thiết lập IAM Users/Groups, MFA, xây dựng Custom VPC với Public/Private Subnets

**Tuần 3:** [Điện toán và Định danh Ứng dụng - EC2 & Instance Profiling](1.3-week3/)
- Khởi chạy EC2, sử dụng IAM Roles, quản lý EBS Volumes

**Tuần 4:** [Lưu trữ và Cơ sở dữ liệu - Cloud9, S3 & RDS](1.4-week4/)
- Thiết lập môi trường Cloud9, triển khai S3 Static Website, khởi tạo RDS MySQL

**Tuần 5:** [Khả năng mở rộng - Lightsail & Auto Scaling](1.5-week5/)
- Làm quen với Docker, triển khai Application Load Balancer và Auto Scaling Groups

**Tuần 6:** [Giám sát và Tự động hóa - CloudWatch & Lambda](1.6-week6/)
- Xây dựng Dashboard, tạo Alarms, viết Lambda functions cho automation

**Tuần 7:** [Quản lý Hệ thống - AWS Systems Manager](1.7-week7/)
- Sử dụng SSM Session Manager, Run Command, quản lý fleet với Tagging

**Tuần 8:** [Infrastructure as Code - CloudFormation & CDK](1.8-week8/)
- Học CloudFormation, triển khai hạ tầng với AWS CDK (TypeScript)

**Tuần 9:** [Tối ưu hóa Mạng và Chi phí - VPC Flow Logs & Cost Optimization](1.9-week9/)
- Phân tích network traffic, sử dụng Cost Explorer và Compute Optimizer

**Tuần 10:** [Kiến trúc Serverless - API Gateway, Lambda & DynamoDB](1.10-week10/)
- Xây dựng RESTful API hoàn chỉnh với serverless architecture

**Tuần 11:** [Quản trị và Tuân thủ - Governance & Well-Architected Review](1.11-week11/)
- Kiểm toán hạ tầng, quản lý Service Quotas, áp dụng Well-Architected Framework

**Tuần 12:** [Dự án Capstone và Chuẩn bị Nghề nghiệp - Career Readiness](1.12-week12/)
- Triển khai dự án tổng hợp, luyện thi chứng chỉ SAA-C03, chuẩn bị portfolio
