Event 2 - AWS Cloud Mastery Series #1
**Summary Report:** “AWS Cloud Mastery Series #1 provided participants with an overview of AI/ML/GenAI on AWS, hands-on demos, and practical guidance for building generative AI solutions.”  

**Event Objectives**  
- Introduce AI/ML and Generative AI landscape on AWS  
- Demonstrate AWS AI/ML services and SageMaker Studio  
- Provide guidance on prompt engineering and RAG integration  
- Enable hands-on experience building a generative AI chatbot  

**Speakers**  
- Kha Van – AWS Vietnam  
- Workshop facilitators from AWS AI/ML team  

**Key Highlights**  
- Date & Time: Saturday, November 15, 2025, 8:30 – 12:00 GMT+7  
- Location: AWS Vietnam Office, Bitexco Financial Tower, Ho Chi Minh City  
- Sessions:  
  - Welcome & Introduction: Registration, networking, workshop overview, ice-breaker  
  - AI/ML Overview: SageMaker platform, data preparation, model training, MLOps, live demo  
  - Generative AI: Amazon Bedrock, foundation models (Claude, Llama, Titan), prompt engineering, RAG integration, Bedrock agents, guardrails, live chatbot demo  

**Takeaways**  
- Understanding AI/ML and GenAI fundamentals  
- Hands-on experience with SageMaker and Bedrock  
- Knowledge of prompt engineering and retrieval-augmented generation  
- Networking and practical insights from AWS experts

Event 4 - AWS Cloud Mastery Series #3
**Summary Report:** “AWS Cloud Mastery Series #3 focused on AWS Well-Architected Security Pillar, covering IAM, detection, infrastructure protection, data protection, and incident response.”  

**Event Objectives**  
- Provide overview of AWS Well-Architected Security Pillar  
- Introduce modern IAM architecture and best practices  
- Demonstrate detection, monitoring, and automation techniques  
- Cover network, workload, data protection, and incident response strategies  

**Speakers**  
- Kha Van – AWS Vietnam  
- Security specialists from AWS  

**Key Highlights**  
- Date & Time: Tuesday, November 25, 2025, 8:30 – 12:00 GMT+7  
- Location: Bitexco Financial Tower, Ho Chi Minh City  
- Sessions:  
  - Security Foundation: Least Privilege, Zero Trust, Defense in Depth, Shared Responsibility Model  
  - IAM: Users, Roles, Policies, SSO, permission sets, MFA, Access Analyzer  
  - Detection & Monitoring: CloudTrail, GuardDuty, Security Hub, EventBridge automation  
  - Infrastructure & Workload Protection: VPC, Security Groups, NACLs, WAF, Shield, Network Firewall, EC2/ECS/EKS security  
  - Data Protection: Encryption, KMS, Secrets Manager, Parameter Store, access guardrails  
  - Incident Response: Playbooks, automation with Lambda/Step Functions, real-world cases in Vietnam  

**Takeaways**  
- Comprehensive understanding of AWS Security Pillar best practices  
- Knowledge of IAM, monitoring, infrastructure, data, and incident response  
- Awareness of automation and security tools to enforce compliance  
- Insights for implementing security in cloud environments
